# CLI Reference

```
loka [command] [flags]
```

## Global Flags

| Flag | Default | Description |
|------|---------|-------------|
| `--server, -s` | active deployment | Control plane address |
| `--token, -t` | | Auth token |
| `--output, -o` | `table` | Output: `table`, `json` |

## Deployments

```bash
loka deploy local --name dev            # Start locally
loka deploy local --name dev -f         # Foreground
loka deploy aws --name prod --region us-east-1 --workers 3
loka deploy gcp --name staging --project my-proj --workers 2

loka deploy list                        # List all deployments (* = active)
loka use prod                    # Switch active deployment
loka deploy rename staging qa           # Rename
loka deploy status                      # Status of active deployment
loka deploy status prod                 # Status of specific deployment
loka deploy down                        # Stop active
loka deploy down dev                    # Stop specific
loka deploy destroy prod                # Remove permanently
```

Deployments are stored in `~/.loka/deployments.json`. All commands (`session`, `exec`, etc.) target the active deployment.

## Sessions

```bash
loka session create --image python:3.12-slim --mode execute
loka session create --image ubuntu:22.04 --mode ask --name my-session
loka session list
loka session get <id>
loka session destroy <id>
loka session pause <id>
loka session resume <id>
loka session mode <id> ask
```

## Commands

```bash
loka exec <session-id> -- echo "hello"
loka exec <session-id> -- python3 script.py
loka exec <session-id> --parallel --cmd "echo A" --cmd "echo B"
```

## Checkpoints

```bash
loka checkpoint create <session-id> --label "before-change"
loka checkpoint list <session-id>
loka checkpoint tree <session-id>
loka checkpoint restore <session-id> <checkpoint-id>
loka checkpoint diff <session-id> <cp-a> <cp-b>
```

## Workers

```bash
loka worker list
loka worker get <id>
loka worker drain <id>
loka worker undrain <id>
loka worker remove <id>
loka worker label <id> gpu=true
loka worker top
```

## Images

```bash
loka image pull ubuntu:22.04
loka image list
```

## Tokens

```bash
loka token create --name my-server --expires 86400
loka token list
loka token revoke <id>
```

## Other

```bash
loka status
loka version
loka completion bash
```
