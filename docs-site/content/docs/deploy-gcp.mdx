# Deploy to GCP

Run LOKA on Google Cloud with nested virtualization for KVM support.

## Quick deploy

```bash
loka deploy gcp --project my-project --zone us-central1-a --workers 2
```

## Manual setup

### 1. Create instances with nested virtualization

```bash
# Control plane
gcloud compute instances create loka-cp \
  --zone=us-central1-a \
  --machine-type=e2-standard-2 \
  --image-family=ubuntu-2204-lts --image-project=ubuntu-os-cloud

# Workers (nested virt required)
gcloud compute instances create loka-worker-1 \
  --zone=us-central1-a \
  --machine-type=n2-standard-8 \
  --min-cpu-platform="Intel Haswell" \
  --enable-nested-virtualization \
  --image-family=ubuntu-2204-lts --image-project=ubuntu-os-cloud
```

### 2. Install and configure

Same as AWS — run the install script, configure, start.

```bash
curl -fsSL https://rizqme.github.io/loka/install.sh | bash
```
